# 第一章：游戏入门

## 1.1 游戏概述

**三国武将录**是一款基于浏览器的三国文字冒险游戏。你扮演一位出身不同的人物，在乱世中做出抉择，最终达成 17 种结局之一。

游戏共分 **4 个阶段**：

| 阶段 | 名称 | 事件数 |
|------|------|--------|
| 第一阶段 | 少年期 | 5 + 1（角色专属） |
| 第二阶段 | 崛起期 | 6 + 1（角色专属） |
| 第三阶段 | 争霸期 | 6 + 1（角色专属） |
| 第四阶段 | 终局期 | 3 + 1（角色专属） |

每阶段结束时（少年期除外），可能触发**危机事件**——处理失败则直接死亡。

## 1.2 属性系统

### 可见属性（在界面中显示）

| 属性 | 含义 | 影响方向 |
|------|------|---------|
| 武 | 武力、战场生存能力 | 战斗相关结局、危机判定 |
| 智 | 谋略、判断力 | 谋士系结局、危机判定 |
| 德 | 品德、道义 | 仁义系结局、民心 |
| 魅 | 魅力、人际影响力 | 人望相关结局、危机判定 |

### 隐藏属性（不在界面显示，影响结局）

| 属性 | 含义 |
|------|------|
| 命运 | 综合运气，影响危机判定和部分结局 |
| 忠义 | 忠诚与义气，影响忠义系结局 |
| 魏 / 蜀 / 吴 | 三大阵营的倾向值，影响阵营系结局 |

> 所有属性值限制在 **[0, 100]** 范围内。

## 1.3 结局判定顺序

游戏结束时按以下顺序判定结局：

1. 若中途触发死亡（危机判定失败 / 选择即死选项），直接进入对应**死亡结局**
2. 若存活至终局，按顺序匹配**正常结局**，第一个满足条件者生效
3. 最后一种结局「平凡一生」无条件匹配，作为兜底

## 1.4 标记系统简介

部分选择会设置**标记（flag）**，解锁后续隐藏选项。例如：少年期「带乡亲撤离」会设置 `farmer_led_villagers` 标记，在崛起期解锁一个额外选项。详见附录。

# 第三章：危机事件必读

> ⚠️ **危机事件**在每个阶段结束时可能触发（少年期免疫）。每种危机都有一个**安全出口**选项，代价较大但保证存活；其他选项需要属性判定，失败直接死亡。

## 🤒 病入膏肓

**事件描述：** 连月征战，殚精竭虑，你终于积劳成疾，一病不起。高烧数日不退，军医束手无策，帐中弥漫着汤药的苦涩气息。部将们面色凝重地守在帐外，窃窃私语间皆是对你性命的担忧。你感……

**✅ 安全出口：**「将后事托付部下，安心养病」
  → 代价：武-3，智-2，命运+15

**🎲 判定选项：**
  - 「寻访名医全力医治」（需 命运≥10）
    失败结果：触发「death_illness」死亡结局
  - 「强撑病体继续征程」（需 武≥25）
    失败结果：触发「death_illness」死亡结局

**💡 攻略建议：** **最常见危机。** 若命运<30或武<25，务必选「将后事托付，安心养病」（安全出口）。拥有 `saved_doctor` 标记时会出现第四个选项，成功率极高，优先选择。平时注意留意能获得 `saved_doctor` 标记的事件。

## 🗡️ 月黑杀机

**事件描述：** 夜半三更，月黑风高。你在睡梦中被一丝异样的气息惊醒，借着微弱的烛光，赫然发现数道黑影已翻墙而入，寒光闪烁的匕首直指你的咽喉。这些训练有素的刺客显然蓄谋已久——你……

**✅ 安全出口：**「舍弃一切财物，从密道逃走」
  → 代价：武-5，魅-3，命运+10

**🎲 判定选项：**
  - 「奋起反抗，与刺客搏斗」（需 武≥30）
    失败结果：触发「death_assassinated」死亡结局
  - 「大声呼救，召来护卫」（需 魅≥15）
    失败结果：触发「death_assassinated」死亡结局

**💡 攻略建议：** 武力高（≥30）的武将优先选「奋起反抗」；魅力高（≥15）的优先选「大声呼救」。商贾之子等武力极低的武将，直接选「从密道逃走」（安全出口），不要赌判定。

## ⚔️ 兵临绝境

**事件描述：** 战场之上，风云突变。敌军以数倍兵力将你团团围困于一处山谷之中，前有重兵把守，后无退路可循。箭矢如雨般倾泻而下，身边的将士一个接一个倒下，你的战马也已中箭倒毙。天……

  - 「拼死杀出一条血路」（需 武≥8）

**💡 攻略建议：** 武力判定门槛仅武≥8，大多数武将可通过。智力流武将（智≥40）可选「以智谋突围」，成功还有额外智力奖励。名声已损无所谓的路线（如枭雄路线）可直接「投降保命」，忠义-8但命运+10。

**事件描述：** 大祸临头——你被朝廷以谋逆之罪逮捕下狱。阴暗潮湿的牢房中，沉重的铁链锁住你的手脚，狱卒冷漠的目光仿佛在看一个将死之人。据闻朝堂之上已有多人上书要求将你斩首示众，……

**💡 攻略建议：** 此危机只在德行/忠义极低时触发，是「你走了坏路」的惩罚。智魅双高（智≥35且魅≥25）可选「慷慨陈词」，成功率高且有属性奖励。「认罪伏法」是最稳的安全出口，代价是武-5、魅-5，但忠义+10，适合想转型的玩家。

# 第五章：全结局路线指南

> 本章为高玩提供每个结局的详细路线建议。死亡结局通过危机判定失败触发，不单独列出路线。

**解锁条件：** 武≥85，智≥85，魅≥80

**推荐武将：** 寒门书生 或 行伍士卒（双线发展）
