# 三国武将录 数值平衡调整设计

日期：2026-04-06
状态：已批准

## 问题概述

游戏当前结局分布严重偏向"平凡一生"（兜底项），玩家几乎无法触发多样化结局。

### 根因分析

| 问题 | 数据 |
|------|------|
| 智力膨胀 | 全局期望增益 +48.6，远超武+31.5、德+26.7 |
| 属性集中在中间区 | 初始值12~40，每阶段均匀增长，终值多在50~70——不够触发高门槛结局 |
| 吴阵营几乎不可达 | 全局仅+3.55期望增益，终局期为0，wu_admiral(吴>=80)几乎不可能 |
| 角色事件无阵营加成 | 24个角色事件的78个选项阵营效果全部为0 |
| 高门槛结局过多 | 多数结局要求某属性>=75~90，但24个事件的随机增长很难让属性突破70+ |

### 调整方向

双管齐下：适度降低结局门槛 + 增大关键选项属性波动 + 补充阵营事件缺口 + 角色阵营种子。

### 目标

每局都有合理概率触发不同特色结局。"平凡一生"仍然是兜底但不应该是绝大多数玩家的默认命运。稀有结局（一统天下、以身殉道）保持珍贵感。

## 第六部分：hermit 条件改造

```json
{
  "德": { "min": 75 },
  "命运": { "min": 62 },
  "no_extreme": true

```

```json
{
  "德": { "min": 68 },
```
