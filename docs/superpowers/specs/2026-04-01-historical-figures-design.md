# 三国知名人物与历史事件融入设计

## 概述

在现有三国武将录游戏中融入 25+ 位历史人物作为 NPC，并引入势力关系系统（魏/蜀/吴），通过改写现有事件 + 新增历史专属事件的方式，让玩家在各阶段体验经典三国故事，与知名人物互动，最终势力倾向影响结局。

## 1. 势力关系系统

### 1.1 新增隐藏属性

在 `characters.json` 的每个角色中新增 3 个隐藏属性，与现有 `命运/忠义` 并列：

| 属性 | 含义 | 初始值 | 范围 |
|------|------|--------|------|
| `魏` | 与曹魏势力的亲疏 | 50 | 0-100 |
| `蜀` | 与蜀汉势力的亲疏 | 50 | 0-100 |
| `吴` | 与东吴势力的亲疏 | 50 | 0-100 |

角色初始偏差（可选，体现身份背景）：
- 寒门书生：`蜀 55`（书生更易被仁德理念吸引）
- 商贾之子：`魏 55`（商业环境与曹魏重实利的风格契合）
- 其余角色：三项均为 50（中立）

### 1.2 机制集成

势力值完全复用现有 `hidden_effects` 和 `hidden_conditions` 机制：

**事件选项中影响势力值**：
```json
{
  "text": "助关羽突围",
  "effects": { "武": 3 },
  "hidden_effects": { "蜀": 8, "魏": -3 },
  "set_flags": ["saved_guanyu"],
  "result": "你与关云长并肩杀出重围..."
}
```

**事件条件用势力值筛选**：
```json
{
  "conditions": { "蜀": { "min": 65 } },
  "description": "诸葛亮主动找到你商议北伐大计..."
}
```

### 1.3 game.js 改动

- `hidden` 对象扩展支持 `魏/蜀/吴` 三个键（与 `命运/忠义` 并列存储）
- `characters.json` 中每个角色的 `hidden` 字段添加 `魏/蜀/吴` 初始值
- `_checkConditions` 和 `_matchesEnding` 已经支持检查 `hidden` 属性的 min/max，逻辑无需修改
- 属性夹值 [0, 100] 的逻辑复用现有代码

### 1.4 人物个人关系（Flag 系统）

通过现有 `set_flags` / `requires_flags` / `excludes_flags` 系统记录与具体人物的关系：

| Flag | 设置时机 | 后续影响 |
|------|----------|----------|
| `befriend_guanyu` | 少年期/崛起期帮助关羽 | 争霸期关羽败走麦城事件解锁特殊选项 |
| `studied_under_zhuge` | 崛起期拜访诸葛亮 | 终局期北伐事件有额外智略选项 |
| `saved_zhaoyun` | 争霸期长坂坡救援赵云 | 后续赵云回报恩情 |
| `served_caocao` | 崛起期投效曹操 | 魏系后续事件有"旧部"身份加成 |
| `allied_sunce` | 崛起期随孙策平江东 | 东吴系事件更信任你 |
| `joined_coalition` | 少年期加入讨董联军 | 崛起期更多主公招揽你 |
| `witnessed_lianbuan` | 崛起期见证连环计 | 争霸期相关事件解锁 |
| `helped_huatuo` | 少年期救助华佗 | 同时设置 `saved_doctor`，兼容现有危机事件逻辑 |

## 2. 历史人物名录

### 2.1 按势力分组（约 25 人）

**曹魏阵营（8 人）**：曹操、荀彧、郭嘉、夏侯惇、典韦、司马懿、张辽、曹丕

**蜀汉阵营（8 人）**：刘备、关羽、张飞、诸葛亮、赵云、黄忠、马超、姜维

**东吴阵营（8 人）**：孙坚、孙策、孙权、周瑜、鲁肃、陆逊、甘宁、太史慈

**中立/其他（5 人）**：吕布、董卓、貂蝉、袁绍、华佗

| 阶段 | 历史时期 | 活跃人物 |

## 3. 事件规划

### 3.1 现有事件改写（10-15 个）

对现有通用事件中涉及三国背景的部分进行改写，将模糊的背景替换为具体历史人物和场景。**保留原有机制和数值平衡**，只改写文本、增加 `hidden_effects`（势力值变化）和相关 flag。

代表性改写：

| 原事件 ID | 原标题 | 改写方向 |
|-----------|--------|----------|
| `youth_01` | 虎牢关前的少年 | 强化吕布战三英描写，选项增加 `joined_coalition` flag |
| `rise_04` | 含"桃园结义" | NPC 具象化为刘备，结拜选项 → `蜀+10, befriend_liubei` |
| `war_01` | 赤壁风云 | 强化周瑜/诸葛亮描写，选择分别倾向不同势力 |
| `final_assassination` | 鸿门之宴 | "新帝"具象为曹丕，增加势力判定 |
| `youth_doctor` | 路遇行医 | 行医者具象为华佗，flag 改为 `helped_huatuo`（兼容原 `saved_doctor`） |

## 5. UI 变化

### 5.1 势力倾向展示

势力值在数据层面存储在 `hidden` 对象中（与 `命运/忠义` 并列），但与命运/忠义不同的是，势力值在游戏 UI 中**可见**。这是有意为之——玩家需要感知势力倾向来做出有意义的阵营选择，而命运/忠义保持隐藏是因为它们代表命运的不可知性。

在属性面板新增"势力倾向"一栏：

- 显示格式：`⚔魏 72 | 🏯蜀 45 | ⛵吴 38`
- 只显示当前数值，选择后的势力值变化不在选项中预览（保留一定的策略探索感）

### 5.2 角色选择界面

- 不显示势力值初始偏差（保持隐藏属性特性）
- 角色描述文本不变

### 5.3 结局画面

- 势力结局展示对应势力的特色描述和图标
- 其余结局不受影响
